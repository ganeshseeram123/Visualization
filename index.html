Visualisation on GDP Dataset
This dashboard provides an interactive exploration of global GDP data, illustrating the economic performance, sector contributions, and their impacts across different countries and years.
Project Overview and Workflow

This project harnesses the power of data visualization to explore and analyze global GDP data. Starting with data abstraction, we collected and cleaned GDP data, segmenting it by country and economic sectors such as agriculture, industry, and services. The transformation process involved normalizing the data to ensure compatibility across different visualization tools.

Using Tableau and D3.js, we crafted interactive visualizations that allow users to engage with the data dynamically. Tableau was utilized for its robust data handling and visualization capabilities, enabling the creation of detailed, interactive charts and maps. D3.js was employed to develop custom visualizations like bubble charts and area plots, providing a deeper level of interaction and customization.

Each visualization aims to provide insights into different aspects of economic performance, from general GDP overviews to detailed sector contributions and trends in agricultural land use. The project is hosted on VizHub, allowing for real-time data interaction and accessibility from anywhere in the world.

This comprehensive approach from data collection to interactive visualization demonstrates the power of modern data science and visualization techniques in making economic data accessible and understandable.
